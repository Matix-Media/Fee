# Fee
Fee is a free, alternative Explorer for the command line of Windows


### Install
To install copy the fee.exe file in the 
C:\Windows
folder.
Or simply use the setup


## Usage
Current folder
```
>> fee
```

local folder
```
>> fee Documents
```

Path
```
>> fee C:\Users\user\Documents
```
